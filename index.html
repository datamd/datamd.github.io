Portfolio

Aneuvis

A web app built using RShiny to quantify numerical chromosomal variation in single cells. Built as part of a PhD project in collaboration between the Mar lab and Montagna lab at Albert Einstein College of Medicine. Watch a 7-minute video tutorial of Aneuvis here.

AbbreviatoR

AbbreviatoR is an RShiny web application for automatically generating an alphabetized list of abbreviations for a thesis or manuscript. Just copy-paste from Word or Excel into the text box! Built as a side project while a PhD student at Albert Einstein College of Medicine.

Oncomix

Oncomix is an R package (available on Bioconductor) for identifying oncogene candidates based on gene expression patterns in tumor and adjacent normal tissue. This software was built as part of a PhD project in the Mar lab.

ReceptLoss

ReceptLoss is an R package for identifying tumor suppressor candidates based on gene expression patterns in tumor and adjacent normal tissue. This software was built as part of a PhD project in the Mar lab at Albert Einstein College of Medicine.
